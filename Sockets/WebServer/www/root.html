You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /multiply -- returns a result of a multiplication between 2 numbers
    • Example: /multiply?num1=2&num2=3
  • /github -- returns information of someone's repository
    • Example: /github?query=users/amehlhase316/repos
  • /compare -- compare two provided numbers
    • Example: /compare?num1=3&num2=4
  • /guess -- returns a result of the /guess game
    • Example: /guess?num=4&max=5 (num is the number you guess, the range is from 0 to max.)
